📄 Real-Time Document Analyzer

🚀 Overview
Real-Time Document Analyzer is an AI-powered application that extracts text from documents, generates concise summaries, and performs sentiment analysis using Natural Language Processing (NLP).

✨ Features
- Extract text from PDF, DOCX, and TXT files
- Generate document summaries
- Perform sentiment analysis
- Support multiple document formats
- User-friendly interface

🛠️ Technologies Used
- Python
- NLP
- PDFPlumber
- TextBlob
- LexRank Summarizer
- Jupyter Notebook
